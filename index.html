Our Tools

We take security as a priority, so we run all of our code through Github. We work with Rails, .Net and if you’re up to it, we also work with Salesforce. Our projects are full of responsiveness thanks to AngularJS, ReactJS and our own custom JS experiences.

We use Agile methodologies for the best organization while managing projects (scrum framework). For this purpose we use Trello, Assembla, Jira or any board that suits you.

If you need to go mobile, we do Hybrid tech: Ionic allows us to deliver iOS and Android apps with our custom servers on Rails, or connecting to your backend API if needed.
